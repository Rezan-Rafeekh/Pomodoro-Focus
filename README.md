# Cozy Focus — Pomodoro Timer 🌸

A minimal **Pomodoro focus timer** designed for studying, working, or reading in a calm environment.
The interface uses **soft cozy shades of pink, brown, and white** with subtle accent highlights to create a relaxing productivity space.

This project is intentionally lightweight and simple, built using **pure HTML, CSS, and JavaScript** without frameworks.

---

## ✨ Features

* ⏱ **Pomodoro Timer**

  * Default 25-minute focus session
  * 5-minute break timer
  * Start, Pause, and Reset controls

* ⚙ **Custom Timer Settings**

  * Adjust study duration
  * Adjust break duration

* 🔊 **Ambient Sounds**

  * Rain
  * Looping background audio for focus

* 📊 **Session Counter**

  * Tracks completed focus sessions during usage

* 🎨 **Cozy Aesthetic UI**

  * Soft pink, brown, and white color palette
  * Accent highlights using `#8b597b`
  * Minimal and distraction-free layout

---

## 🛠 Tech Stack

This project uses very simple web technologies:

* **HTML5** – structure
* **CSS3** – styling and layout
* **JavaScript (Vanilla)** – timer logic and audio controls

No frameworks or package managers are required.

## ▶️ Running the Project

No installation is required.

1. Download or clone the repository.
2. Open the project folder.
3. Double-click:
index.html
The application will open in your browser.

---

### Optional (Recommended)

If using **VS Code**, install the **Live Server** extension.

Then:

1. Right-click `index.html`
2. Click **Open with Live Server**

This will automatically reload the page whenever you update the code.
